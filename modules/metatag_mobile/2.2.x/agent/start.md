# metatag_mobile — agent start

Metatag submodule: adds ~34 mobile/browser-UI meta tags — `viewport`, `theme-color`,
Apple `apple-mobile-web-app-*`, and the Microsoft `msapplication-*` tile/pinned-site
family. Depends on `metatag`. Set via any Metatag defaults form.

- The mobile / UI tags → [configure/tags.md](configure/tags.md)
