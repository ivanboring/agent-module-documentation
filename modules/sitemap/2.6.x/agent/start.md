# sitemap — agent start

Builds a **human-readable** HTML overview page (default `/sitemap`) from pluggable
sections (front page, menus, vocabularies). NOT the XML `sitemap.xml` module.
Config UI: **Admin → Config → Search and metadata → Sitemap** (`sitemap.settings`).
Submodules: `sitemap_book`, `sitemap_metatag`.

- Settings form (page title, path, message, enabled sections, CSS) → [configure/settings.md](configure/settings.md)
- Sitemap section plugin type + built-ins (Frontpage/Menu/Vocabulary) → [plugins/sitemap.md](plugins/sitemap.md)
- Blocks (Sitemap, Syndicate) → covered in [configure/settings.md](configure/settings.md)
- Alter taxonomy listings via hooks → [hooks/hooks.md](hooks/hooks.md)
- Templates & theme hooks → [theming/theming.md](theming/theming.md)
- Permissions → [permissions/permissions.md](permissions/permissions.md)
