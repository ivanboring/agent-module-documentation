# Dublin Core Advanced tags

Adds ~40 qualified Dublin Core terms (each `<meta name="dcterms.NAME">`), grouped with the
basic Dublin Core set. Highlights: `dcterms.abstract`, `dcterms.accessRights`,
`dcterms.license`, `dcterms.rightsHolder`, `dcterms.created`, `dcterms.modified`,
`dcterms.issued`, `dcterms.alternative`, `dcterms.bibliographicCitation`,
`dcterms.audience`, `dcterms.educationLevel`, `dcterms.isPartOf`/`hasPart`,
`dcterms.isVersionOf`/`hasVersion`, `dcterms.replaces`/`isReplacedBy`,
`dcterms.requires`/`isRequiredBy`, `dcterms.conformsTo`, `dcterms.provenance`,
`dcterms.spatial`, `dcterms.temporal`, `dcterms.medium`, `dcterms.extent`,
`dcterms.tableOfContents`, `dcterms.valid`, accrual-* and reference-* terms.

Requires `metatag_dc`. Schema: `config/schema/metatag_dc_advanced.metatag_tag.schema.yml`.
