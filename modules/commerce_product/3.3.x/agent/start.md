# commerce_product — agent start

Commerce submodule: the catalog layer. Defines `commerce_product` (marketing item),
`commerce_product_variation` (purchasable, priced, SKU unit), and product **attributes**
(Size/Color) that generate variations. Requires `commerce`, `commerce_price`,
`commerce_store`. Config hub: **/admin/commerce/config** →
Product types / Variation types / Attributes (`commerce_product.configuration`).

- Product types, variation types, attributes, permissions → [configure/catalog.md](configure/catalog.md)
