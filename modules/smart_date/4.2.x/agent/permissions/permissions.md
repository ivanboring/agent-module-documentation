# Permissions

From `smart_date.permissions.yml`:

- `administer smart date formats` — access the Smart date formats admin UI to create/edit/
  delete `smart_date_format` config entities (`restrict access: true` — trusted role only).

The **smart_date_recur** submodule adds: `make smart dates recur`, `reschedule smart date
recur instances`, `cancel smart date recur instances` — see that submodule's docs.
