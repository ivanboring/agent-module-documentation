# video_embed_media — agent start

Submodule of `video_embed_field`. Provides a Media **source** plugin so external videos become
core Media entities usable in the Media library. Depends on `media` + `video_embed_field`.

- Set up the media type / source plugin → [configure/media-source.md](configure/media-source.md)
- Migrate these entities to core oEmbed → see the `vem_migrate_oembed` submodule docs.
