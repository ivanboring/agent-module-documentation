# Permissions

Static (`field_permissions.permissions.yml`):
- **`administer field permissions`** — manage field permissions and settings (choose the
  permission type / grid on fields). `restrict access: TRUE`.
- **`access private fields`** — view and edit the stored values of all fields set to
  **Private**, for any user's entities. `restrict access: TRUE`.

Dynamic: each field whose type is **Custom** contributes its own permissions, generated by
`permission_callbacks: FieldPermissionsService::getAllPermissions`:
- `create <field>`
- `edit own <field>` / `edit <field>`
- `view own <field>` / `view <field>`

These appear on **People → Permissions**, grouped per field. Access checks run through
`FieldPermissionsService::getFieldAccess()` (invoked from core `hook_entity_field_access`).
