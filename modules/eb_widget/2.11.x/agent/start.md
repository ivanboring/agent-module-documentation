# dropzonejs_eb_widget — agent start

Submodule of **dropzonejs**. Provides Entity Browser widget plugins that use the
`dropzonejs` upload element. Machine name `dropzonejs_eb_widget`; depends on
`dropzonejs`, core `file`, `entity_browser`. No admin route of its own — widgets are
configured inside an Entity Browser (`/admin/config/content/entity_browser`).

- Widget plugins, their config keys & the inheritance chain → [configure/widget.md](configure/widget.md)
