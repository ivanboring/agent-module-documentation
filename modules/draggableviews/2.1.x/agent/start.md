# draggableviews — agent start

Adds a draggable Views field + weight sort so editors reorder view rows by drag-and-drop.
Order is stored per view/display/args in the `draggableviews_structure` table. Requires `views`.

- Set up a draggable+sortable view → [configure/views-setup.md](configure/views-setup.md)
- Helper class, structure table & migrate destination → [api/helpers.md](api/helpers.md)
- Permission that gates the "Save order" button → [permissions/permissions.md](permissions/permissions.md)
