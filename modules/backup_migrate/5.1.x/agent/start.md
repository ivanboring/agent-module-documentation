# backup_migrate — agent start

Backs up/restores the Drupal DB and files and migrates sites between environments.
No module deps. Config UI: **Admin → Config → Development → Backup and Migrate**
(`/admin/config/development/backup_migrate`, configure route `backup_migrate.quick_backup`).

- Sources, destinations, schedules, settings profiles (config entities) → [configure/config.md](configure/config.md)
- Define custom source/destination plugins → [plugins/source-destination.md](plugins/source-destination.md)
- Drush commands (quick backup, schedules, listings) → [drush/commands.md](drush/commands.md)
- Permissions → [permissions/permissions.md](permissions/permissions.md)
