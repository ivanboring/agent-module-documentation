# menu_block — agent start

Adds a `menu_block` block plugin (derived per menu, extends core `SystemMenuBlock`) with rich
tree/level/label options. Depends on core `menu_ui`. Configure per-block at
**Admin → Structure → Block layout** → *Place block* → "Menus" category. No admin page, no
permissions, no drush.

- Block settings (levels, depth, follow, fixed parent, dynamic title) → [configure/block-settings.md](configure/block-settings.md)
- Theme hook suggestions & templates → [theming/suggestions.md](theming/suggestions.md)
