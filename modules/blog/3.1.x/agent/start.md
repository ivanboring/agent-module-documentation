# Blog — agent index

Multi-user blogging (formerly Drupal core). Enabling `blog` installs a `blog_post`
content type, a `blog` View with page/block/feed displays, per-user blog pages, and a
"My blog" account-menu link. No settings form (`configure` is null); access uses standard
node bundle permissions.

- **Content model, Views, routes & permissions** → [configure/blog.md](configure/blog.md)
- **Services, helper functions & hooks the module implements** → [api/blog.md](api/blog.md)
