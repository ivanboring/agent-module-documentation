# rh_media — agent start

Rabbit Hole integration for core **Media**. Registers a `RabbitHoleEntityPlugin`
(`entityType = "media"`), so the "Rabbit Hole settings" tab appears on media type and media
edit forms. No config or API of its own — behavior comes from the base module.

- Configure media behaviors (action, redirect, override) → [../../../rabbit_hole/1.2.x/agent/configure/settings.md](../../../rabbit_hole/1.2.x/agent/configure/settings.md)
- Per-entity-type permissions (`rabbit hole administer media`, `rabbit hole bypass media`) → [../../../rabbit_hole/1.2.x/agent/permissions/permissions.md](../../../rabbit_hole/1.2.x/agent/permissions/permissions.md)
