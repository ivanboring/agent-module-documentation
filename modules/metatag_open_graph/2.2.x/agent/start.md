# metatag_open_graph — agent start

Metatag submodule: adds the full Open Graph (`og:*`, `article:*`, `book:*`, `profile:*`,
`video:*`, `place:*`) tag family (~59 tags) for rich social sharing previews. Depends on
`metatag`; pair with `metatag_facebook` / `metatag_twitter_cards`.

- The Open Graph tags → [configure/tags.md](configure/tags.md)
