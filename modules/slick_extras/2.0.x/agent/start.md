# slick_extras — agent start

Add-on for the **slick** carousel module (depends on `slick:slick >= 3.x`). The base module
registers a `SlickSkin` plugin (`slick_extras_skin`) that adds six extra CSS skins, and the
project bundles example (`slick_example`) and development (`slick_devel`) learning sub-modules.
No config UI, permissions, routes, or Drush of its own — it is sample/scaffolding code the
maintainers suggest you clone and uninstall in production. Skins appear in the Slick optionset
**Skin** dropdown after cache clear.

- Extra skins it registers + how to add your own → [plugins/slick_extras.md](plugins/slick_extras.md)
