# media_entity_browser — agent start

Ships prebuilt Entity Browser config (a browser + View + image style + embed button) that
lets editors pick/embed core Media. Mostly configuration; no PHP services, permissions, or
plugins. Requires `entity_browser`, `inline_entity_form`, core `media` + `views`.

- What config it installs and how to wire it up → [configure/browsers.md](configure/browsers.md)
- Grid styling library + preprocess hook → [theming/view.md](theming/view.md)
