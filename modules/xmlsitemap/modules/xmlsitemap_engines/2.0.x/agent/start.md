# xmlsitemap_engines — agent start

Submodule of xmlsitemap. Pings/submits the sitemap to search engines after regeneration.
Requires `xmlsitemap`. Config UI: **Config → Search and metadata → XML Sitemap → Search
Engines** (`/admin/config/search/xmlsitemap/engines`, route `xmlsitemap_engines.settings`,
permission `administer xmlsitemap`).

- Settings (engines, custom URLs, interval) → [configure/settings.md](configure/settings.md)
- Register an engine via hook → [hooks/hooks.md](hooks/hooks.md)
