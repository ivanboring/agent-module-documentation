# xmlsitemap — agent start

Builds sitemaps.org-compliant `sitemap.xml` from an incrementally-indexed link table. Config
UI: **Admin → Config → Search and metadata → XML Sitemap** (`/admin/config/search/xmlsitemap`,
route `xmlsitemap.admin_search`). Submodules: `xmlsitemap_custom`, `xmlsitemap_engines`.

- Settings, per-bundle inclusion, sitemaps/contexts → [configure/settings.md](configure/settings.md)
- Add/read links & generate in code (services) → [api/services.md](api/services.md)
- Contribute links / reshape output (hooks) → [hooks/hooks.md](hooks/hooks.md)
- Drush regenerate/rebuild/index → [drush/commands.md](drush/commands.md)
- Permissions → [permissions/permissions.md](permissions/permissions.md)
