# video_embed_wysiwyg — agent start

Submodule of `video_embed_field`. Adds a CKEditor 5 button + text-format filter for inserting
provider videos into rich text. Depends on `ckeditor5`, `editor`, `field`, `video_embed_field`.

- Enable the filter + toolbar button on a text format → [configure/ckeditor.md](configure/ckeditor.md)
