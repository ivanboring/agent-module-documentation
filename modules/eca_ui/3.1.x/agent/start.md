# eca_ui — agent start

The admin UI for ECA. Registers ECA as a `modeler_api` model owner; models live at
`/admin/config/workflow/eca`. Needs a modeler (e.g. `bpmn_io`) for graphical editing.
Depends on `eca`.

- Admin screens, settings form (`eca.settings`), log view → [configure/admin.md](configure/admin.md)
