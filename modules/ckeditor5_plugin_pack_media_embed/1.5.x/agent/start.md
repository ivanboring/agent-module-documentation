# ckeditor5_plugin_pack_media_embed — agent start

Submodule of CKEditor 5 Plugin Pack. Enables CKEditor 5's **Media Embed** feature — embed
external media (YouTube/Vimeo/oEmbed providers) by pasting a URL. Registered via
`ckeditor5_plugin_pack_media_embed.ckeditor5.yml`. Depends on `ckeditor5_plugin_pack`.

Enable, then add the Media Embed button to a text format toolbar at **Admin → Config → Content
authoring → Text formats and editors** and ensure the format's allowed HTML permits the embed
markup. No config schema, no standalone admin page.
