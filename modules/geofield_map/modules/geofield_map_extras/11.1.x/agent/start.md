# geofield_map_extras — agent start

Submodule of **geofield_map**. Adds two lightweight Google Map field formatters for a
`geofield` Point: a static image map and an iframe embed map. Depends on `geofield_map`
(reuses its Google Maps API key). Not enabled by default; no own settings form.

- Static and embed Google Map formatters (plugin ids, settings, templates) → [configure/formatters.md](configure/formatters.md)
- Theme hooks / templates → [api/theming.md](api/theming.md)
- Parent module (widget, dynamic formatter, Views style, API key) → ../../../../11.1.x/agent/start.md
