# paragraphs_features — agent start

Usability add-ons for the Paragraphs widget (add-in-between, delete confirm, drag & drop,
single-action button) plus a CKEditor 5 "Split paragraph" tool. Requires `paragraphs` +
core `field`/`ckeditor5`. Most features toggle per field-widget; one global setting at
**Admin → Config → Content authoring → Paragraphs features** (`paragraphs_features.settings_form`).

- Enable per-field widget features + the global button setting → [configure/features.md](configure/features.md)
- Enable the CKEditor 5 Split paragraph button → [configure/split-paragraph.md](configure/split-paragraph.md)
