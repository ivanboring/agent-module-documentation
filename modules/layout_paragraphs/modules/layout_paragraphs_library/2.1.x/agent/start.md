# layout_paragraphs_library — agent start

Submodule of **layout_paragraphs**. Integrates the Layout Paragraphs builder with the
Paragraphs Library (`paragraphs_library`), adding **Promote to library** / **Unlink from
library** to the component form. Pure form-alter/AJAX hooks — no config, permissions, or
services of its own. Depends on `layout_paragraphs` and `paragraphs:paragraphs_library`.

- Set up & use reusable library components → [configure/layout_paragraphs_library.md](configure/layout_paragraphs_library.md)
