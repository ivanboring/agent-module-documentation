# Pinterest tags

Adds a "Pinterest" MetatagTag group:

- `pin:description` — Rich Pin description.
- `pin:id` — Rich Pin identifier.
- `pin:media` — the pinnable image/media URL.
- `pin:url` — canonical pin URL.
- `pinterest` — site-ownership verification tag.

Set as Metatag defaults or per entity; token-enabled (map `pin:media` to an image field).
Schema: `config/schema/metatag_pinterest.metatag_tag.schema.yml`.
