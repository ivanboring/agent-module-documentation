# Verification tags

Adds a "Site Verification" MetatagTag group (`<meta name="…">`). Paste the token each
service gives you into global Metatag defaults:

- `google-site-verification` (Google Search Console)
- `msvalidate.01` (Bing Webmaster Tools)
- `yandex-verification`, `baidu-site-verification`
- `facebook-domain-verification`, `p:domain_verify` (Pinterest)
- `norton-safeweb-site-verification`, `pocket-site-verification`
- `siwecostoken`, `zoom-domain-verification`

Schema: `config/schema/metatag_verification.metatag_tag.schema.yml`.
