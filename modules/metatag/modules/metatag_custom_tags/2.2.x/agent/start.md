# metatag_custom_tags — agent start

Metatag submodule: define your own meta tags (no code) as `metatag_custom_tag` config
entities; they then appear in every Metatag form. Depends on `metatag`. Config UI route
`entity.metatag_custom_tag.collection`. Adds permission `administer custom meta tags`.

- Defining & managing custom tags → [configure/custom-tags.md](configure/custom-tags.md)
