# commerce_store — agent start

Commerce submodule. Defines the `commerce_store` content entity and `commerce_store_type`
config entity — the container that owns products/orders and sets default currency, billing
countries, address and timezone. Requires `commerce` + `commerce_price`. Admin:
**/admin/commerce/config/store** (`entity.commerce_store.collection`).

- Create/manage stores & store types → [configure/stores.md](configure/stores.md)
