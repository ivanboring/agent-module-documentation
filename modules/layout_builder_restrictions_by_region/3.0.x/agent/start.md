# layout_builder_restrictions_by_region — agent start

Submodule of `layout_builder_restrictions`. Adds the `entity_view_mode_restriction_by_region`
restriction plugin so allowed blocks can be set **per region** of each allowed layout.
Requires `layout_builder` + `layout_builder_restrictions`. No permissions of its own.

- Enable + configure per-region rules → [configure/by-region.md](configure/by-region.md)
