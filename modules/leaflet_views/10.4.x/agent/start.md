# leaflet_views — agent start

Views integration for Leaflet. Adds Views plugins that map a result set of Geofield entities.
Depends on `leaflet` + `views`. Configured entirely in the Views UI (no admin page).

- Views style/row/attachment plugins & how to build a map view → [plugins/views-plugins.md](plugins/views-plugins.md)
- Alter features/map settings (hooks) → [hooks/hooks.md](hooks/hooks.md)
