# superfish — agent start

Renders any menu as a multi-level jQuery-Superfish dropdown/flyout via a **Superfish block**
(block plugin `id: superfish`, derived per menu). No global config UI (`configure` null) — you
configure each block when placing it at **Admin → Structure → Block layout**. External assets
come from the Composer library `lobsterr/drupal-superfish`.

- Place & configure a Superfish block (all settings keys) → [configure/block.md](configure/block.md)
- Templates / theme overrides → [theming/templates.md](theming/templates.md)
- Alter the menu tree manipulators via hook → [hooks/hooks.md](hooks/hooks.md)

No permissions, drush, or plugin types of its own.
