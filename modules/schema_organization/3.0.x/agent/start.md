# schema_organization — agent start

Submodule of [schema_metatag]. Adds a Metatag group `schema_organization` (`@type`
Organization / LocalBusiness tree) with Tag plugins for Organization properties (name,
description, logo, image, address, geo, telephone, contactPoint, openingHoursSpecification,
priceRange, acceptsReservations, menu, sameAs, aggregateRating, starRating, brand, memberOf,
additionalType, potentialAction, url, @id). Usually configured once globally under the site's
**Metatag** settings; values render as JSON-LD via schema_metatag. No config UI of its own.
