# search_api_solr_legacy — agent start

Compatibility connectors for old Solr (3.6, 4.5+, 5.x). Enable, then pick the legacy connector
on the Search API server form. Requires `search_api_solr`. New sites should not use this.

- Legacy SolrConnector plugins & config templates → [plugins/legacy-connectors.md](plugins/legacy-connectors.md)
