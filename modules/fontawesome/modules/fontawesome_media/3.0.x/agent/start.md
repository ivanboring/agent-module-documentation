# fontawesome_media — agent start

Submodule of `fontawesome`. Registers a `font_awesome_icon` media source so Font Awesome
icons can be stored/reused as media entities. Requires `fontawesome` + core `svg_image`.
No config UI beyond creating a Media type that uses this source.

- Create the icon media type & add icons → [configure/media-source.md](configure/media-source.md)
