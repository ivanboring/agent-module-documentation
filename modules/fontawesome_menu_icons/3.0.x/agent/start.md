# FontAwesome Menu Icons — agent index

Attaches a Font Awesome icon to a Drupal menu link. No admin settings form and **no
configure route**: configuration is per menu link. Depends on the `fontawesome` module
(loads the icon font) and core `menu_ui`.

- **Add / set an icon on a menu link** (form fields, config storage, drush) →
  [configure/menu-link-icons.md](configure/menu-link-icons.md)
- **Where icons are stored & the render/hook surface** (link `options` keys, config
  object, hooks) → [api/icon-options.md](api/icon-options.md)
