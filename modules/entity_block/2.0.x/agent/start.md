# entity_block — agent start

Provides one derivable block plugin (`entity_block`, deriver `EntityBlockDeriver`) that
renders any content entity in a chosen view mode as a block. One derivative is created per
content entity type that has a view builder. No admin settings form, permissions, services,
or submodules — placement is done wherever blocks are placed.

- Place an Entity Block, pick the entity + view mode, in Block layout or Layout Builder → [configure/entity_block.md](configure/entity_block.md)
