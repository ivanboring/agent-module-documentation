# editor_advanced_link — agent start

Adds extra `<a>` attributes (title, class, id, aria-label, rel, target) to the CKEditor
link dialog. Ships as a CKEditor 5 plugin ("Advanced links") configured per **text format**;
no admin route of its own. Depends only on core `editor`.

- Enable & configure per text format (allowed attributes, filter interplay) → [configure/text-format.md](configure/text-format.md)
- Integrations & extension points (Linkit, Editor File, CKEditor 4→5 upgrade, form alter) → [extend/integrations.md](extend/integrations.md)
