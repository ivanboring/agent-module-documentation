# views_bulk_operations_example — agent start

Demo submodule with one sample VBO action (`ViewsBulkOperationExampleAction`) exercising
preconfiguration, configuration form, access, and result messages. Depends on
`views_bulk_operations`. Enable on a dev site as a template.

For how to write your own action, see the main module doc:
`modules/views_bulk_operations/4.4.x/agent/plugins/vbo-action.md`. No other capabilities.
