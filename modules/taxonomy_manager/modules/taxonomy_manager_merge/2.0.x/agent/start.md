# taxonomy_manager_merge — agent start

Submodule of **taxonomy_manager**. Adds a **Merge terms** operation to the Taxonomy Manager
tree editor: fold selected duplicate terms into one target term, reassigning their references.
Depends on `taxonomy_manager` + contrib **Term Merge** (`term_merge`) and
**Term Reference Change** (`term_reference_change`). No config or permissions of its own.

- The Merge terms toolbar button, modal & merge flow → [configure/configure.md](configure/configure.md)
- Parent module → [../../../../2.0.x/agent/start.md](../../../../2.0.x/agent/start.md)
