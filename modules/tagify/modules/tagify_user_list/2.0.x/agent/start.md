# tagify_user_list — agent start

Submodule of **tagify**. Field widget
`tagify_user_list_entity_reference_autocomplete_widget` renders user references as Tagify
tags with avatar + info label. Requires `tagify`. Set on the field's Manage form display.

- Widget settings (avatar field, image style, matching) → [configure/widget.md](configure/widget.md)
- Autocomplete matcher service + alter hook → [api/api.md](api/api.md)
