# tagify — agent start

Tag-chip widgets (Tagify JS lib) for entity-reference fields. Global config route
`tagify.settings` at `/admin/config/tagify/settings`; per-field settings on each widget.
Depends on core `taxonomy`.

- Global + per-field widget settings → [configure/settings.md](configure/settings.md)
- Services (autocomplete matcher, hierarchical terms, theme resolver) → [api/services.md](api/services.md)
- Alter autocomplete matches → [hooks/hooks.md](hooks/hooks.md)
- Render elements (`entity_autocomplete_tagify`, `select_tagify`) → [theming/render-elements.md](theming/render-elements.md)
