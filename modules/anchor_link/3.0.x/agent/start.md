# anchor_link — agent start

Bundles the `vardot/ckeditor5-anchor-drupal` CKEditor 5 plugin: adds an "Anchor link"
toolbar button + improved link dialog. Depends on core `ckeditor5` + `editor`. No admin
config page of its own — configured per **text format** at `/admin/config/content/formats`.

- Enable the button + allow anchor markup in a text format → [configure/text-format.md](configure/text-format.md)
- Linkit matcher + CKEditor 4→5 upgrade plugin → [extend/integrations.md](extend/integrations.md)
