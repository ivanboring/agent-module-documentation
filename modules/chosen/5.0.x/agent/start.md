# chosen — agent start

Applies the Chosen JS library to `<select>` elements site-wide for searchable, tag-style
dropdowns. Requires the `chosen_lib` submodule (assets). Config UI: **Admin → Config → User
interface → Chosen** (`/admin/config/user-interface/chosen`, route `chosen.admin`).

- Settings form + all config keys (thresholds, selector, text) → [configure/settings.md](configure/settings.md)
- How Chosen attaches to elements + JS libraries → [theming/attach.md](theming/attach.md)
- Migrate D7 Chosen settings → [extend/migrate.md](extend/migrate.md)
