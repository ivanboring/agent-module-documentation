Installs a set of example webforms specifically built for reviewing and testing form accessibility (labels, containers, wizards, and advanced patterns).

---

Webform Examples Accessibility ships demo webforms curated to exercise accessibility concerns so you can review and test how Webform renders for assistive technology. On install it imports five forms: basic, advanced, containers, labels, and wizard accessibility examples. A small hook implementation and a CSS library adjust presentation to highlight accessibility-relevant markup. Use these forms to audit ARIA attributes, label associations, fieldset/legend grouping, keyboard navigation, and screen-reader behavior, or as a reference for building your own accessible forms. It depends only on `webform` and provides no permissions, schema, or config UI. Intended for development, QA, and accessibility-review sites rather than production.
