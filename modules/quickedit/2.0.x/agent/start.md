# quickedit — agent start

In-place (front-end) field editing via contextual links + AJAX. Contrib continuation of core's
former Quick Edit. Depends on core `contextual`, `editor`, `field`, `filter`. No config UI
(`configure` null) — works once enabled and the user has the permission.

- In-place editor plugin type & bundled editors → [plugins/in-place-editors.md](plugins/in-place-editors.md)
- Services (editor selector, metadata, access) → [api/services.md](api/services.md)
- Alter editor selection / field render via hooks → [hooks/hooks.md](hooks/hooks.md)
- Permission → [permissions/permissions.md](permissions/permissions.md)
