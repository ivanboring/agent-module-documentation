# page_manager — agent start

Engine for custom pages: `page` config entities (path + parameters + access) rendered through
selectable `page_variant` display variants. Can override core routes. API/engine only — enable
the **page_manager_ui** submodule for the Structure → Pages wizard. Depends on `block`, `ctools`.

- Page / page_variant config entities & how routing works → [configure/pages.md](configure/pages.md)
- Display variants it ships + add your own → [plugins/display-variants.md](plugins/display-variants.md)
- Contexts & the context event (add contexts in code) → [api/contexts.md](api/contexts.md)
- Permissions → [permissions/permissions.md](permissions/permissions.md)
