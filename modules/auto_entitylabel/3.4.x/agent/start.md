# auto_entitylabel — agent start

Generates an entity's label from a token pattern and can hide the label field. Config lives
on a per-bundle **Automatic label** tab (link template `auto-label`), e.g.
`/admin/structure/types/manage/<type>/auto-label`. No central config route.

- Configure a bundle's label behavior & pattern → [configure/settings.md](configure/settings.md)
- Per-entity-type permissions → [permissions/permissions.md](permissions/permissions.md)
- Enable custom entity types & alter labels in code → [hooks/hooks.md](hooks/hooks.md)
