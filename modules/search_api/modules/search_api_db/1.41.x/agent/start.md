# search_api_db — agent start

Submodule of **search_api**. Provides one Search API backend plugin, `search_api_db`
(`Drupal\search_api_db\Plugin\search_api\backend\Database`), that indexes into SQL tables.
No own permissions, routes, or Drush — configured through Search API's server UI.

- Create a Database server & backend options → [configure/backend.md](configure/backend.md)
