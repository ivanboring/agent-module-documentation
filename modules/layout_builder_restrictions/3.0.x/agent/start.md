# layout_builder_restrictions — agent start

Controls which blocks/categories/layouts editors may use in Layout Builder, per view mode.
Defines the **LayoutBuilderRestriction** plugin type; ships the `entity_view_mode_restriction`
plugin. Requires `layout_builder`. Global admin page + per-view-mode UI.

- Configure restrictions (per view mode + plugin admin page) → [configure/settings.md](configure/settings.md)
- Write a restriction plugin (the plugin type) → [plugins/restriction.md](plugins/restriction.md)
- Permission that gates config → [permissions/permissions.md](permissions/permissions.md)
