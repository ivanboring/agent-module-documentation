# fontawesome_iconpicker_widget — agent start

Submodule of `fontawesome`. Adds field widget `fontawesome_iconpicker_widget` (extends the
base FA widget) giving a visual iconpicker for `fontawesome_icon` fields. Requires
`fontawesome`. No config UI — enable per field via Manage form display.

- Enable the widget & download the JS library → [configure/widget.md](configure/widget.md)
