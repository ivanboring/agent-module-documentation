# eca_form — agent start

Form API in ECA: **events** (build/alter, validate, submit for any form), **conditions**
(operation, field exists/value, submitted/triggered, has-errors), **actions** (add fields,
set label/description/weight/default/options/access/required/disabled/#states, set errors,
form-state get/set, rebuild, redirect, build entity). Registers into ECA Core; no new plugin
types. Depends on `eca`.
