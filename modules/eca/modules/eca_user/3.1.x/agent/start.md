# eca_user — agent start

User accounts in ECA: **events** (login/logout/register/cancel, etc.), **conditions**
(current/other user id, role, permission), **actions** (load current user, new user, switch
account / service account / switch back, get preferred langcode). Registers into ECA Core; no
new plugin types. Depends on `eca`.
