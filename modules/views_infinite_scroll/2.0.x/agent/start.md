# views_infinite_scroll — agent start

Adds an **Infinite Scroll** Views pager plugin (`infinite_scroll`): a "Load more" button or
auto-load-on-scroll instead of numbered pages. Depends on `views`; the View must have **AJAX
enabled**. Configured entirely inside the View's *Pager* settings (no admin page, no permissions).

- Enable & configure the Infinite Scroll pager in a View → [configure/views_infinite_scroll.md](configure/views_infinite_scroll.md)
- Theme the pager / override the JS → [theming/views_infinite_scroll.md](theming/views_infinite_scroll.md)
