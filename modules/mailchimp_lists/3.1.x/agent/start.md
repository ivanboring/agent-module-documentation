# mailchimp_lists (Mailchimp Audiences) — agent start

Mailchimp submodule. Ties Drupal entities to Mailchimp audiences via a subscription field
type. Requires `mailchimp`, core `field` + `options`. Overview UI:
`/admin/config/services/mailchimp/lists` (route `mailchimp_lists.overview`). Merge-field
mapping: `/admin/config/services/mailchimp/fields`.

- Audiences overview, subscription field, merge vars, webhooks → [configure/audiences.md](configure/audiences.md)
