# maxlength — agent start

Adds a character limit + live countdown to field widgets. No admin settings page,
no permissions — configured per widget on **Structure → Content types → [type] →
Manage form display** (open a widget's gear icon → "MaxLength Settings"). Core `^10 || ^11`.

- Configure limits on a field widget (settings keys, soft vs hard) → [configure/widget-settings.md](configure/widget-settings.md)
- Add MaxLength support to a custom widget (hook) → [hooks/hooks.md](hooks/hooks.md)
- Frontend behavior (library, countdown message, CKEditor 5) → [theming/frontend.md](theming/frontend.md)
