# field_group — agent start

Groups fields into tabs/fieldsets/details/HTML elements on entity **form** and **view**
displays. Configured through core Field UI (no dedicated settings route). Depends on core
`field`. Groups are stored as third-party settings on the form/view display config entity.

- Create & configure groups in the UI → [configure/field-groups.md](configure/field-groups.md)
- Add a new group format (plugin type) → [plugins/field-group-formatter.md](plugins/field-group-formatter.md)
- Alter grouping/render via hooks → [hooks/hooks.md](hooks/hooks.md)
- Theme groups (templates, render elements) → [theming/theming.md](theming/theming.md)
