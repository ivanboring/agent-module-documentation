# context_ui — agent start

Admin UI submodule of **context** (`part_of: context`). Adds the screens to build/manage
`context` config entities (conditions + reactions). No config entities or schema of its own —
it edits the parent module's `context` entity. Depends on `context:context`. Release
candidate (5.0.0-rc2).

- Where the UI lives, routes, permission, Context Inspector block → [configure/context_ui.md](configure/context_ui.md)

See the parent module for the entity/plugin/API details: `modules/context/5.0.x/agent/start.md`.
