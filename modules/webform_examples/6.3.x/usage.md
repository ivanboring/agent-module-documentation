Installs a library of ready-made demonstration webforms showcasing Webform elements and advanced features, usable as live demos or cut-and-paste starting points.

---

Webform Examples is a config-only module (no PHP) that ships a set of fully built example webforms exercising Webform's more advanced capabilities. On install it imports demo forms including computed elements (static and AJAX), element `#states` conditional logic, flexbox layouts, input masks, a multi-step wizard, a style guide, and an optional cards form (imported when the Webform Cards submodule is present). These forms are meant for demonstrating and testing functionality, learning how a feature is configured, or copying YAML/element definitions into your own forms. It depends only on `webform` and provides no permissions, schema, services, or code — everything lives under `config/install` and `config/optional`. Enable it on a development or demo site, disable it in production once you've harvested what you need.
