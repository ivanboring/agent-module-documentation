# slick_views — agent start

Adds Slick carousel **Views style plugins** ("Slick Carousel" + "Slick Grouping"). No admin
page — configured per-View in the Views UI (Format). Requires `views` + `slick` (which supplies
the optionsets and library). Options stored in the View's config; schema in
`config/schema/slick.views.schema.yml`.

- Use the carousel/grouping Views styles + their options → [plugins/views-style.md](plugins/views-style.md)
