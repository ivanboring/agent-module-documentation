# jquery_ui_autocomplete — agent start

Library-provider module. Ships the jQuery UI Autocomplete widget removed from core as the
Drupal asset library `jquery_ui_autocomplete/autocomplete`. Depends on `jquery_ui` and
`jquery_ui_menu` (the suggestion list is a menu). No config, permissions, or plugins.

- Attach/depend on the autocomplete asset library → [api/library.md](api/library.md)
