# asset_injector — agent start

Add custom CSS/JS to page output via config entities with visibility conditions. No
dependencies. Admin UI: **Admin → Config → Development → Asset Injector**
(`/admin/config/development/asset-injector`, route `asset_injector.development`); separate CSS
and JS collections.

- CSS/JS asset entities, conditions, JS options, config export → [configure/assets.md](configure/assets.md)
- Alter the generated library via hook → [extend/hooks.md](extend/hooks.md)
- Permissions (security-sensitive) → [permissions/permissions.md](permissions/permissions.md)
