# fullcalendarview_generator — agent start

Submodule of `fullcalendar_view`. Adds one Drush command that scaffolds a "Full Calendar Display"
View from content types + date fields. Depends on `views` and `fullcalendar_view`. No UI, no
permissions, no config schema. Enable: `drush en fullcalendarview_generator -y`.

- The `fullcalendarview:generate` (`fcvg`) command → [drush/fullcalendarview_generator.md](drush/fullcalendarview_generator.md)
- Parent module → [../../../../5.2.x/agent/start.md](../../../../5.2.x/agent/start.md)
