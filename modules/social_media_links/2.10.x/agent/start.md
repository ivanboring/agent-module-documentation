# social_media_links — agent start

Provides the `social_media_links_block` block: icon links to ~50 social networks, rendered
with a selectable icon set. Two plugin types make it extensible. Place & configure via
**Structure → Block layout** (no dedicated settings route). Field variant lives in the
`social_media_links_field` submodule.

- Place & configure the block (platforms, iconset, appearance) → [configure/block.md](configure/block.md)
- Add a new social network → [plugins/platform.md](plugins/platform.md)
- Add a new icon set → [plugins/iconset.md](plugins/iconset.md)
