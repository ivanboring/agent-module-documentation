# selective_better_exposed_filters — agent start

Extends Better Exposed Filters: adds "Show only used items" (and related) options to exposed
Views filters so they list only values present in the result set. No config page, no
permissions — configured per filter in the Views UI. Requires `better_exposed_filters`.

- Enable & tune the options in the Views UI → [configure/options.md](configure/options.md)
- How it works / overridden BEF widgets → [extend/widgets.md](extend/widgets.md)
