Hidden helper (package Migration, `hidden: true`) for the `migrate_example_advanced` module.

destination content type, fields, taxonomy vocabulary and comment type the advanced migration
writes into, plus REST configuration. Depends on core `comment`, `image`, `text`, `taxonomy`,
`rest`. Pure scaffolding, enabled automatically as a dependency — nothing runnable, no
config/plugins to document.
