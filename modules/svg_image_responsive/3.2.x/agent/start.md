# svg_image_responsive — agent start

Submodule of **svg_image**. Overrides the core **Responsive image** formatter
(`responsive_image`) so SVGs render through responsive image displays. Depends on
`svg_image` + core `responsive_image`. No new UI (`configure` null), no plugin types.

- What it overrides & the SVG formatter settings → [extend/overrides.md](extend/overrides.md)
