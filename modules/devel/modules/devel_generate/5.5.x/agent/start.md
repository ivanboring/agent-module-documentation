# devel_generate — agent start

Submodule of Devel. Bulk-generates dummy entities (nodes, users, terms, vocabs, menus,
media, block content). UI under **Config → Development → Generate**
(`/admin/config/development/generate/*`). Requires `devel`.

- Generate forms & options (UI) → [configure/generate.md](configure/generate.md)
- Drush `devel-generate:*` commands → [drush/commands.md](drush/commands.md)
- Add a custom generator (plugin type) → [plugins/devel-generate.md](plugins/devel-generate.md)
- Alter generated values via hook → [hooks/hooks.md](hooks/hooks.md)
- Permissions → [permissions/permissions.md](permissions/permissions.md)
