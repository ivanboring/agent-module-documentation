# quick_node_clone — agent start

Adds a Clone action to nodes; deep-copies fields + Paragraphs into a new node edit form.
Depends on core `node`. Config UI: **Config → Quick Node Clone**
(`/admin/config/.../quick-node-clone`, route `quick_node_clone.settings`).

- Settings (title prefix, excluded fields, clone status) → [configure/settings.md](configure/settings.md)
- Alter the cloned node / paragraph fields (hooks) → [extend/hooks.md](extend/hooks.md)
- Per-content-type clone permissions → [permissions/permissions.md](permissions/permissions.md)
