# file_browser тАФ agent start

Ships a ready-made, Media-library-style **Entity Browser for files**: a Masonry grid of
existing images/files to select + a DropzoneJS upload tab. It is configuration on top of
Entity Browser, Entity Embed, and DropzoneJS тАФ no admin settings form of its own. Depends on
`entity_browser`, `entity_embed`, `dropzonejs` (+ `dropzonejs_eb_widget`), and core `field`,
`file`, `image`, `views`, `node`, `menu_ui`, `path`, `text`.
