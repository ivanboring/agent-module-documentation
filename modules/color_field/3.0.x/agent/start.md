# color_field — agent start

Adds a `color_field_type` field (hex + optional opacity) with several picker
widgets and display formatters. Configured entirely via core Field UI; depends
on core `field`. Also provides Token, Feeds, and migrate integrations.

- Field storage/settings, widgets & formatters → [configure/field.md](configure/field.md)
- Swatch templates & theme hooks → [theming/templates.md](theming/templates.md)
- Color value objects (Hex/RGB/HSL conversion) in code → [api/color-classes.md](api/color-classes.md)
