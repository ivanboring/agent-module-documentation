# ds (Display Suite) — agent start

Region-based layouts + configurable field templates + virtual custom fields for any entity
view display. Builds on core Layout API (`layout_discovery`). Admin UI:
**Admin → Structure → Display Suite** (`/admin/structure/ds`, route `ds.structure`); per-entity
layout is chosen on each **Manage display** screen.

- Configure layouts, custom fields, global settings → [configure/settings.md](configure/settings.md)
- Define DsField / DsFieldTemplate plugins → [plugins/plugins.md](plugins/plugins.md)
- Alter fields, templates, layouts via hooks → [extend/hooks.md](extend/hooks.md)
- Permissions → [permissions/permissions.md](permissions/permissions.md)
- Submodules (extras, switch view mode, devel) → [configure/submodules.md](configure/submodules.md)
