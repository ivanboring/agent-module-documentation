# Display Suite permissions

From `ds.permissions.yml`:

| Permission | Gates |
|---|---|
| `admin display suite` | Access to Display Suite settings pages generally. |
| `admin fields` | Manage DS custom fields for every entity (restricted — trusted roles only). |
| `admin classes` | Manage the region/layout CSS classes list. |

Submodule `ds_extras` adds its own permissions (see `configure/submodules.md`), including
switching a field's UI display and using the "fields extra" features.
