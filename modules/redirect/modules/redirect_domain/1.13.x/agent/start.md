# redirect_domain — agent start

Submodule of Redirect. Redirects whole domains (optionally per sub-path) to a destination,
via a request subscriber. Depends on `redirect`. Config form route
`redirect_domain.domain_list` (`/admin/config/search/redirect/domain`), gated by
`administer redirects`.

- Configure domain redirects → [configure/domain-redirects.md](configure/domain-redirects.md)
