# easy_email_commerce — agent start

Integration submodule of **easy_email**. Adds `commerce_order` email tokens and declares the
Commerce order-receipt email so you can send Commerce order emails from Easy Email templates.
Depends on `easy_email`, `commerce_order`, `commerce_price`. No entities, permissions, routes, or
services of its own. Part of the `easy_email` project (`drupal/easy_email`).

- Send/build a Commerce order email; tokens & the declared receipt → [configure/easy_email_commerce.md](configure/easy_email_commerce.md)
