# Year Only — agent index

One field type, `yearonly`, storing only a year as an integer. No admin settings page,
no permissions, no services — everything is configured on the field itself via Field UI
or config entities.

- **Add / configure the field** (range settings, widget, formatter): [configure/field-setup.md](configure/field-setup.md)
- **Programmatic surface** (field type internals, range helper, sample values, Feeds target): [api/field-type.md](api/field-type.md)
