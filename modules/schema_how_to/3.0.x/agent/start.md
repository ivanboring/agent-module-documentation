# schema_how_to — agent start

Submodule of [schema_metatag]. Adds a Metatag group `schema_how_to` (`@type` HowTo) with Tag
plugins for HowTo properties (name, description, step, supply, tool, estimatedCost, image, url,
@id). Configure the tags with tokens under the site's **Metatag** settings; values render as
JSON-LD via schema_metatag. No config UI of its own.
