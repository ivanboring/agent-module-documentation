# search_api_solr_autocomplete — agent start

Solr suggester plugins for the Search API Autocomplete module. Enable per search display in
Search API Autocomplete settings, then choose a suggester. Requires `search_api_autocomplete`
+ `search_api_solr`.

- Suggesters (Terms, Spellcheck, Suggester) + hooks/events → [plugins/suggesters.md](plugins/suggesters.md)
