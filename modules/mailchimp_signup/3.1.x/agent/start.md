# mailchimp_signup (Mailchimp Signup) — agent start

Mailchimp submodule. Configurable newsletter signup forms as blocks and/or pages. Requires
`mailchimp`. Admin UI: `/admin/config/services/mailchimp/signup` (route
`mailchimp_signup.admin`).

- Create signup forms, choose display (block/page), audiences → [configure/signup.md](configure/signup.md)
- Permissions → [permissions/permissions.md](permissions/permissions.md)
