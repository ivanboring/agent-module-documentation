# coffee — agent start

Alfred/Spotlight-style admin quick-search overlay (keyboard shortcut → type page name →
navigate). No module dependencies. Config UI: **Admin → Config → User interface → Coffee**
(`/admin/config/user-interface/coffee`, route `coffee.configuration`).

- Configure searched menus + max results (`coffee.configuration`) → [configure/settings.md](configure/settings.md)
- Add custom commands / Views results to the box via a hook → [hooks/hooks.md](hooks/hooks.md)
- Permissions → [permissions/permissions.md](permissions/permissions.md)
