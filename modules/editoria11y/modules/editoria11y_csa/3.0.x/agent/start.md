# editoria11y_csa — agent start

Editoria11y addon: contrast + developer test suites, custom-test config entities, crawler
dashboard, license tooling. Requires `editoria11y` (and `key` for license storage). Config UI:
`/admin/config/content/editoria11y/csa` (route `editoria11y_csa.settings`); custom tests at
`/admin/config/content/editoria11y/custom-tests`. Reuses `administer editoria11y checker`
permission.

- Configure contrast/developer tests + custom-test entities → [configure/settings.md](configure/settings.md)
- License management Drush commands → [drush/commands.md](drush/commands.md)
