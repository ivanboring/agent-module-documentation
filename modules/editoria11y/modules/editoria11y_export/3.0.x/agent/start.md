# editoria11y_export — agent start

Thin Editoria11y submodule: installs filterable **Views Data Export** Views so dashboard
reports can be downloaded as CSV, and attaches export links to the Editoria11y dashboard.
Requires `editoria11y` and `views_data_export`. No PHP logic, no settings form.

- Configure/customize the export Views → [configure/exports.md](configure/exports.md)
