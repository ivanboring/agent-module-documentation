# Permissions

Defined in `trash.permissions.yml` (plus dynamic per-entity-type permissions from
`TrashPermissions::entityTypePermissions`).

| Permission | Gates |
|---|---|
| `administer trash` | Configure which entity types/bundles use the trash bin (settings form) and full access to the trash listing. Trusted permission. |
| `access trash` | View the contents of the trash bin at `/admin/content/trash`. |
| `view deleted entities` | View individual soft-deleted entities. |
| dynamic `restore <type> entities` / `purge <type> entities` (per enabled entity type) | Restore / purge trashed entities of that type. Generated by `TrashPermissions::entityTypePermissions`. |

The trash listing routes require `access trash+administer trash` (either grants access).

```
drush role:perm:add content_editor 'access trash'
```
