# geocoder_geofield — agent start

Geocoder ⇄ contrib **Geofield** integration. Adds a Geofield preprocessor/field plugin,
`Geometry` + `Geohash` dumpers, file geometry providers (GeoJSON/GPX/KML) and geofield
formatters. Configured through **geocoder_field**'s per-field settings (pick the Geofield as
source/target, use the Geometry dumper). Depends on `geocoder_field` + `geofield`.

- What it adds & how to use it → [configure/geofield.md](configure/geofield.md)
