# webform_options_custom — agent start

Adds a **Custom options** webform element that turns HTML/SVG markup into a clickable
single/multiple selector (seating charts, maps, floor plans). Reusable templates are
`webform_options_custom` config entities. Depends only on `webform`. Ships config schema;
no permissions/drush. Manage templates at `/admin/structure/webform/options/custom/manage`.

- Element, templates & config entity → [configure/webform_options_custom.md](configure/webform_options_custom.md)
