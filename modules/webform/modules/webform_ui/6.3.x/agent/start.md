# webform_ui — agent start

The build UI for webforms: add/edit/reorder/change element types via drag-and-drop, plus a
Source (YAML) editor. Depends only on `webform`; defines no permissions/plugins/services of
its own (reuses core Webform's). Element manager at
`/admin/structure/webform/manage/{webform}`.

- Element builder & source editor routes/forms → [configure/webform_ui.md](configure/webform_ui.md)
