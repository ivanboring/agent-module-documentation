# webform_image_select — agent start

Adds a `webform_image_select` element (visual, image-based options) plus a `webform_image_select_images` config entity for reusable image sets. Depends on `webform`. Ships config schema; management uses the `administer webform` permission (no own permissions.yml). Images UI: **Admin → Structure → Webform → Options → Images** (`/admin/structure/webform/options/images/manage`).

- Use the image-select element in a form → [plugins/webform_image_select.md](plugins/webform_image_select.md)
- Create/manage reusable image sets (config entity) → [configure/webform_image_select.md](configure/webform_image_select.md)
- Alter image lists in code → [hooks/webform_image_select.md](hooks/webform_image_select.md)
