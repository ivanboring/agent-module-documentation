# webform_share — agent start

Exposes webforms for embedding on external sites via iframe or a `share.js` snippet,
rendered standalone through a dedicated theme negotiator/display variant. Depends only on
`webform`. No permissions/drush/config schema of its own. Admin embed tab at
`/admin/structure/webform/manage/{webform}/share`.

- Embed endpoints, admin Share tab & isolation → [configure/webform_share.md](configure/webform_share.md)
