# smart_date_recur — agent start

Submodule of **smart_date**. Adds RRULE recurrence to `smartdate` fields (needs
`smart_date` + core `datetime`; uses `simshaun/recurr`). Enable recurrence in the Smart Date
widget's "Repeat" option; manage instances at **Admin → Content → Smart date recurring**
(`/admin/content/smart_date_recur/...`).

- Rule/override entities, formatters, instance management → [configure/recurrence.md](configure/recurrence.md)
- Work with rules in code → [api/rules.md](api/rules.md)
- Drush: prune invalid rules → [drush/commands.md](drush/commands.md)
- Permissions → [permissions/permissions.md](permissions/permissions.md)
