# paragraphs_library — agent start

Submodule of **paragraphs**. Adds a reusable `paragraphs_library_item` content entity so one
authored paragraph can be referenced from many hosts via the bundled **From library**
paragraph type. Depends on `paragraphs`, core `views`, and `entity_usage`. Overview at
`admin/content/paragraphs`; settings route `paragraphs_library_item.settings`.

- Configure the library, reuse workflow & permissions → [configure/library.md](configure/library.md)
