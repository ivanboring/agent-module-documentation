# views_conditional — agent start

Adds one Views field handler, "Views: Views Conditional" (`views_conditional_field`), that
outputs text from an IF/THEN/ELSE condition comparing an earlier field in the row against a
value. Depends only on core `views`. No admin settings form, no permissions, no config
entity — all settings live inside the view. Package `Views`.

- Add & configure the conditional field (operators, tokens, field ordering) → [configure/views_conditional.md](configure/views_conditional.md)
