# simple_sitemap_engines — agent start

Submodule of `simple_sitemap`. Submits sitemaps to search engines on cron and notifies
IndexNow-compatible engines on entity changes. Config UI:
`/admin/config/search/simplesitemap/engines` (route `simple_sitemap.engines.settings`).

- Configure engines, IndexNow, submission interval → [configure/settings.md](configure/settings.md)
