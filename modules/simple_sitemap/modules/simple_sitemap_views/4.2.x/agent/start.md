# simple_sitemap_views — agent start

Submodule of `simple_sitemap`. Adds Views pages (and their argument variations) to XML
sitemaps via a Views display extender + a `views` URL generator plugin. Config UI:
`/admin/config/search/simplesitemap/views` (route `simple_sitemap.views`).

- Include view displays & indexed arguments in sitemaps → [configure/settings.md](configure/settings.md)
