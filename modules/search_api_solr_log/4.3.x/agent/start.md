# search_api_solr_log — agent start

Logs Drupal system events into Solr with a Views + Facets report. Requires `search_api_solr`,
`views`, `facets` (facets_exposed_filters). Config route: `search_api_solr_log.settings`
(`/admin/config/search/search-api-solr-log`).

- Settings, log report & clearing logs → [configure/settings.md](configure/settings.md)
