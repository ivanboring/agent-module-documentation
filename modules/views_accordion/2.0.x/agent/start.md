# views_accordion — agent start

Adds a **jQuery UI accordion** Views **style/format** plugin (id `views_accordion`) that renders
each view row as an accordion section, using the first field as the header/trigger. Depends on
core `views` + contrib `jquery_ui_accordion`. Requires the **Fields** row style and ≥2 fields.
No admin settings form, permissions, or services of its own — all config lives in the view.

- Configure the accordion style + all its options → [configure/views_accordion.md](configure/views_accordion.md)
- Theme/override the accordion markup → [theming/views_accordion.md](theming/views_accordion.md)
