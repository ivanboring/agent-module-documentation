# media_library_form_element — agent start

Provides a Form API render element `#type => 'media_library'` wrapping core Media Library.
Requires core `media_library`. No admin config. Element:
`Drupal\media_library_form_element\Element\MediaLibrary`.

- Use the `media_library` element in a form (properties, return value) → [api/element.md](api/element.md)
- Template & theme hook for the selected-items UI → [theming/template.md](theming/template.md)
