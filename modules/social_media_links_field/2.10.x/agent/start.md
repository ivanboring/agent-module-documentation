# social_media_links_field — agent start

Submodule of **social_media_links**: exposes the platform/iconset system as a field type
(`social_media_links_field`) so social profiles are stored per entity. Depends on
`social_media_links` (for Platform + Iconset plugins). Add via Field UI (`/admin/structure`).

- Add & configure the field (widgets, formatter, settings, theming) → [configure/field.md](configure/field.md)
