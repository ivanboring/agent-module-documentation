# contact_block — agent start

Adds a **Contact block** block plugin that embeds a core Contact form (site-wide or personal)
into any block region. Requires core `block` + `contact`. No settings page, no permissions —
configured when placing the block in **Structure → Block layout**.

- Place & configure the block (form + form display) → [configure/block.md](configure/block.md)
- The block plugin & personal-form behavior → [plugins/contact-block.md](plugins/contact-block.md)
