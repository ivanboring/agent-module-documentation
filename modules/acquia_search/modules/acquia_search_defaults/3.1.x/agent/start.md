# acquia_search_defaults — agent start

Submodule of **acquia_search**. Ships only optional config — a default Search API Solr index
(`acquia_search_index`) on the `acquia_search_server`, plus a search View
(`views.view.acquia_search`) — so a connected site gets working node search out of the box.
No code, no settings, no permissions, no Drush.

- What it installs and how to enable it → [configure/setup.md](configure/setup.md)
- Parent module → [../../../../3.1.x/agent/start.md](../../../../3.1.x/agent/start.md)
