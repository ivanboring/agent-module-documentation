# webform_toggles — agent start

Not an installable submodule in webform 6.3.x. The `modules/webform_toggles/` directory
contains only `js/webform_toggles.element.js` (a `Drupal.behaviors.webformToggle` behavior
wrapping the jQuery Toggles plugin to render checkbox inputs as on/off switches) and has no
`.info.yml`. No PHP, routes, permissions, config, services, or plugins. Legacy asset stub —
nothing to configure.
