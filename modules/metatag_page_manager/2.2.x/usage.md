Lets you attach and configure meta tags on Page Manager pages/variants, so custom landing pages built with Page Manager get proper SEO tags.

---

Page Manager (from CTools) builds custom pages and variants that aren't standard entity canonical routes, so they don't automatically pick up entity meta tags. This submodule integrates Metatag with Page Manager by adding meta tag configuration to page variants, letting site builders set title/description/OG tags on those custom pages with token support. Depends on both Page Manager and the main Metatag module. Useful for landing pages, dashboards, and other Page Manager-driven routes.

---

- Add meta tags to a Page Manager landing page.
- Set a custom title/description on a Page Manager variant.
- Give Panels-built pages proper Open Graph tags.
- Configure SEO for non-entity custom routes.
- Use tokens in meta tags on Page Manager pages.
- Override defaults for a specific page variant.
- Provide social-share previews for campaign landing pages.
- Set canonical URLs on custom pages.
- Control robots indexing per Page Manager variant.
- Keep landing-page SEO in exportable config.
- Differentiate meta tags across multiple variants of one page.
- Ensure dashboards/custom routes aren't missing meta tags.
- Standardize SEO across Page Manager pages.
- Combine with Open Graph/Twitter submodules on custom pages.
- Manage marketing landing-page metadata without code.
- Apply per-variant selection-condition-aware tags.
