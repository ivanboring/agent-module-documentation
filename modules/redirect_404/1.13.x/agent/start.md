# redirect_404 — agent start

Submodule of Redirect. Logs 404s to a SQL store and offers a **Fix 404 pages** report to turn
frequent misses into redirects. Depends on `redirect` + `views`. Config/report route
`redirect_404.fix_404` (`/admin/config/search/redirect/404`).

- Fix-404 report, settings & ignore/API → [configure/fix-404.md](configure/fix-404.md)
- Permissions → [permissions/permissions.md](permissions/permissions.md)
