# addtoany — agent start

Adds AddToAny social share/follow buttons to content entities (node, media, comment) as a
pseudo-field and via blocks. Depends on core `node`. Config UI: **Admin → Config → Web
services → AddToAny** (`/admin/config/services/addtoany`, route `addtoany.admin_settings`).

- Settings form + config keys (buttons, entities, universal button) → [configure/settings.md](configure/settings.md)
- Share block + follow block → [configure/blocks.md](configure/blocks.md)
- Theme hooks, templates, positioning the pseudo-field → [theming/templates.md](theming/templates.md)
- Extend supported entity types via alter hook → [hooks/hooks.md](hooks/hooks.md)
- Permissions → [permissions/permissions.md](permissions/permissions.md)
