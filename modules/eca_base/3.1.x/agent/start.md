# eca_base — agent start

Foundational ECA plugins: cron/custom **events**, scalar/list/token/state **conditions**, and
generic **actions** (tokens, state, key-value, tempstore, lists, lock, translate). Registers into
ECA Core's managers; defines no new plugin types. Depends on `eca`.
