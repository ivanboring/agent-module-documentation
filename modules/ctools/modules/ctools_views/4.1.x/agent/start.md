# ctools_views — agent start

Experimental submodule of `ctools`. Enhances the core Views **Block** display so placed view
blocks gain per-instance settings (items/offset, pager, field hide/reorder, exposed
filters/sorts). Depends on `block` + `views`. No admin UI. Core `^9.5 || ^10 || ^11`.

- How the enhanced Views block display works & its settings → [configure/views-block.md](configure/views-block.md)
