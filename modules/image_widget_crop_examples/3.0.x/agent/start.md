# image_widget_crop_examples — agent start

Demo/reference submodule of image_widget_crop.
Installs a "Crop test" content type, crop types, image styles and displays showing the crop
widget with plain image fields, Media, Entity Browser and Inline Entity Form.

Not for production — enable to explore a working setup, then uninstall. Pulls in `media`,
`responsive_image`, `entity_browser`, `inline_entity_form`, `ctools`. No config UI of its own;
everything is delivered as installed config.
