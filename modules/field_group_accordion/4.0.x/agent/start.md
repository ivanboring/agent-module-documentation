# field_group_accordion — agent start

**Deprecated** (jQuery UI is EOL — prefer Details/Tabs from `field_group`). Submodule of
Field Group that adds two group formats: **Accordion** (container) and **Accordion item**
(section), rendered with the core `jquery_ui_accordion` widget. Depends on `field` +
`jquery_ui_accordion`. No settings route — configured inline via core Field UI.

- Add & configure accordion groups → [configure/accordion.md](configure/accordion.md)
