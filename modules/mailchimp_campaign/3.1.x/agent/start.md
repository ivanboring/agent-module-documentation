# mailchimp_campaign (Mailchimp Campaign) — agent start

Mailchimp submodule. Create/send/test/stat Mailchimp campaigns as Drupal entities. Requires
`mailchimp_lists` + core `filter`. Overview UI:
`/admin/config/services/mailchimp/campaigns` (route `mailchimp_campaign.overview`).

- Manage campaigns: create, embed entities, send, test, stats → [configure/campaigns.md](configure/campaigns.md)
- Permissions → [permissions/permissions.md](permissions/permissions.md)
