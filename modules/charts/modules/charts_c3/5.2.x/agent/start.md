# charts_c3 — agent start

Library submodule of **charts**: registers the **C3.js** backend (Chart plugin `id: c3`).
Depends on `charts`; pulls in C3.js 0.7.20 + D3 7.9.0 (CDN by default). Enable it, then select
"C3" as the library. No config route of its own — configured through Charts settings / per
View / per block.

- Enable C3.js, select it, and manage its CDN/local library → [configure/charts_c3.md](configure/charts_c3.md)
