# charts_blocks — agent start

Submodule of **charts**. Adds block plugins so you can place a chart without a View, entering
the data by hand. Depends on `charts`; needs a library submodule enabled to render. No config
route of its own — configure per block instance in Block layout / Layout Builder / Canvas.

- Place & configure the chart blocks (`charts_block`, `charts_canvas_block`) → [configure/charts_blocks.md](configure/charts_blocks.md)
