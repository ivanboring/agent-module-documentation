# charts_chartjs — agent start

Library submodule of **charts**: registers the **Chart.js** backend (Chart plugin
`id: chartjs`). Depends on `charts`; pulls in Chart.js ^4.4 + date-fns adapter + datalabels
plugin (CDN by default). Adds a `polarArea` chart type. Fully MIT-licensed. Enable it, then
select "Chart.js" as the library. No config route of its own.

- Enable Chart.js, select it, and manage its CDN/local library → [configure/charts_chartjs.md](configure/charts_chartjs.md)
