# charts_billboard — agent start

Library submodule of **charts**: registers the **Billboard.js** backend (Chart plugin
`id: billboard`). Depends on `charts`; pulls in Billboard.js 3.10.3 + D3 7.9.0 (CDN by
default). Enable it, then select "Billboard.js" as the library. No config route of its own —
configured through Charts settings / per View / per block.

- Enable Billboard.js, select it, and manage its CDN/local library → [configure/charts_billboard.md](configure/charts_billboard.md)
