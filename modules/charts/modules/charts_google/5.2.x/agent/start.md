# charts_google — agent start

Library submodule of **charts**: registers the **Google Charts** backend (Chart plugin
`id: google`). Depends on `charts`; loads the Google Charts loader.js (`google/charts` v45)
from Google's servers. Adds a `table` chart type. Enable it, then select "Google" as the
library. No config route of its own.

- Enable Google Charts, select it, and manage its library loader → [configure/charts_google.md](configure/charts_google.md)
