# charts_highcharts — agent start

Library submodule of **charts**: registers the **Highcharts** backend (Chart plugin
`id: highcharts`). Depends on `charts`; pulls in Highcharts 12.5.0 + many official add-on
modules (CDN/jsDelivr by default). Widest chart-type coverage; adds `solidgauge` and
`dumbbell` types and a ColorChanger form. **Commercial license required for commercial use.**
Enable it, then select "Highcharts" as the library. No config route of its own.

- Enable Highcharts, select it, manage its CDN/local libraries + license → [configure/charts_highcharts.md](configure/charts_highcharts.md)
