# geolocation_google_maps — agent start

Adds Google Maps as a map provider + geocoder for the Geolocation module. Requires
`geolocation`. Set the API key at **Admin → Config → Services → Geolocation → Google Maps**
(`/admin/config/services/geolocation/google_maps`, route `geolocation_google_maps.settings`).

- API key, Maps API params hook, geocoders, nested submodules → [configure/setup.md](configure/setup.md)
