# Lightning Roles — agent index

Submodule of Lightning Core. Auto-creates per-content-type user roles from **content-role
templates**. Machine name `lightning_roles`; depends on `node`. Config UI at
`/admin/config/system/lightning/roles` (route `lightning_roles.settings`).

- **Content-role templates & how deployment works**:
  [configure/lightning_roles.md](configure/lightning_roles.md)

No permissions, plugin types, hooks API, or Drush commands of its own.
