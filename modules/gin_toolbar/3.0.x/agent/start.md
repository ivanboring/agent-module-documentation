# gin_toolbar — agent start

Companion module for the **Gin** admin theme that renders Gin's toolbar/navigation on the
front end. No config UI, no permissions, no plugins, no services to call — it works purely
through theme overrides and library attachments once Gin is the active admin/default theme.
Requires `drupal/gin`. Core `^11.2`.
